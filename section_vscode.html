Visual Studio Code

Some useful tips for using VS Code.

keybindings | shortcuts | 自定义快捷键

First, press 'Cmd' + 'K' and 'Cmd' + 'S'

Click upper right corner: Open Keyboard Shortcurs (JSON) to open `keybindings.json` json.

[
    {
        "key": "ctrl+e p",
        "command": "editor.emmet.action.wrapWithAbbreviation",
        "args": {
            "abbreviation": "p"
        },
        "when": "editorHasSelection && editorTextFocus"
    }
]
